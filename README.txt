Implementação do Algoritmo de Earley na linguagem Java
Trabalho para a disciplina Linguística Computacional
Autor: Robson Feitosa

Para execução do experimento é necessário o ambiente JDK 1.7.0 previamente
 instalado e devidamente configurado. Em seguida, basta acessar o diretório 
 raiz: "algoritmo-earley" e executar os comandos (no terminal Linux):

1. javac src/*.java
2. cp aires-treino.parsed src/
3. cd src
4. java Validacao

Dependendo do desempenho do seu processador o resultado pode demorar
algumas horas. Os logs do experimento ficam armazenados na pasta:
"algoritmo-earley/src/experimentos".
